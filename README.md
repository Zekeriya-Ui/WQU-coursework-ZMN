# WorldQuant University Coursework Website

This project is a single-page HTML portfolio for Zacharia Maganga Nyambu's WorldQuant University coursework.

## What is inside
- A dashboard-style homepage inspired by the WorldQuant University course interface.
- Three main course areas:
  - MScFE program details and grades.
  - Applied Data Science Lab projects.
  - Deep Learning Fundamentals Lab projects.
- A summary section showing the final cumulative average.

## File to open
Open `index.html` in a browser to view the website.

## How to use it
- Double-click `index.html`, or open it in a browser.
- Edit the text directly in the HTML file if you want to change titles, descriptions, or links.
- If you want to publish it online, upload `index.html` to GitHub Pages.

## Customization ideas
- Replace placeholder cards with screenshots or certificate images.
- Add links to GitHub projects, reports, or certificates.
- Change colors, fonts, or layout to match your personal branding.
- Add more sections such as contact details, research highlights, or a resume link.

## Notes
- The site is static, so it does not need a backend.
- It works best on GitHub Pages, Netlify, or any static hosting service.
